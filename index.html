This is a private command-line tool developed to manage and interact with multiple YouTube accounts under one interface. The application is used by the developer to automate certain repetitive tasks such as subscribing to specific channels or commenting on targeted videos.

Purpose

This tool leverages the YouTube Data API v3 and Google OAuth2.0 authentication to connect multiple Google accounts via a desktop-based CLI app.

Use Case

  • Connect and authenticate multiple YouTube accounts
  • Subscribe all connected accounts to a specific YouTube channel
  • Post comments across accounts on a specified YouTube video

Privacy

The application is not public and is only used internally by the developer. All data stays on local storage, and the app does not share or expose user data to third parties.
